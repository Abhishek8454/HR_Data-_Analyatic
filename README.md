# HR_Data-_Analyatic
HR Data Analytics Project focuses on analyzing employee-related data to gain meaningful insights that support better HR decision-making. This project explores key factors such as employee attrition, performance, job satisfaction, department-wise distribution, and salary trends. Using Python and data analysis libraries, the dataset is cleaned, processed, and visualized to identify patterns and correlations within the workforce. The project includes exploratory data analysis (EDA) and basic visualizations to present findings in a clear and understandable manner. Overall, this project demonstrates practical skills in data handling, analysis, and visualization applied to real-world HR data scenarios.
